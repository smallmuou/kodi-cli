# kodi-cli

> 命令行集合，方便管理、查看

## 安装

```
git clone 
cd kodi-cli
chmod +x ./install.sh
sudo ./install.sh
```

## 使用

```
starnet:kodi-cli $ kd -h
usage:  kd [OPTIONS] COMMAND [--help]
    
    OPTIONS:
        -h show this help message and exit 
        -u uninstall kd-cli 

    COMMANDS:
        ap current
        ap next
        ap open
        ap pause
        ap play
        ap prev
        ap properties
        ap repeat
        ap shuffle
        ap speed
        apl add
        apl clear
        apl list
        app mute
        app quit
        app vol
        sys reboot
```
